Load Image:
- Click the "Load Image" button on the left side.
- A file chooser dialog will appear.
- Select an image file in the following formats: PPM, PNG or JPEG.
- The loaded image will be displayed in the center.

Save Image:
- Click the "Save Image" button on the left side.
- A file chooser dialog will appear.
- Choose a location and enter a file name and format from the following: PPM, PNG or JPEG.

Brighten/darken:
- Click the "Brighten" button on the right side.
- Enter an integer value in the dialog box.
- Enter a positive value to brighten and a negative value to darken.
- The displayed image will update.
